# dominzaredead 

Another new tool

This tool will grab all the subdomains in a target website

It uses assetfinder and httprobe 

# Install 

      linux@user ~ $ git clone https://github.com/Hirukshacoder/dominzaredead 
      linux@user ~ $ cd dominzaredead 
      linux@user ~ $ ./main.sh
